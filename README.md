# Customer-Behavior-Analysis

##  Project Overview
This project analyzes customer shopping behavior using transactional data to uncover insights that help improve sales, customer engagement, and business decision-making.

The objective is to understand how different factors such as demographics, discounts, product categories, and customer loyalty influence purchasing patterns.

---

##  Business Problem
A retail company wants to understand customer purchasing behavior to improve sales performance, customer satisfaction, and long-term loyalty.

Key questions:
- How do discounts impact purchasing behavior?
- Which customer segments generate the most revenue?
- What factors drive repeat purchases and engagement?
- How can marketing and product strategies be optimized?

---

##  Tools & Technologies
- **Python (Pandas, NumPy)** → Data Cleaning & Feature Engineering  
- **MySQL** → Data Analysis & Querying  
- **Power BI** → Data Visualization & Dashboard  
- **VS Code** → Development Environment  

---

## ⚙️ Project Workflow

1. **Data Preparation (Python)**
   - Cleaned dataset and handled missing values  
   - Created new features like:
     - Customer Segment
     - Engagement Score
     - Customer Lifetime Value (CLV)
     - Discount Impact  

2. **Database Integration**
   - Connected Python to MySQL using SQLAlchemy  
   - Uploaded cleaned dataset into MySQL  

3. **SQL Analysis**
   - Performed business analysis using SQL queries:
     - Revenue by gender  
     - Customer segmentation  
     - Discount impact  
     - Subscription behavior  
     - Top products  

4. **Visualization (Power BI)**
   - Built an interactive dashboard showing:
     - KPIs (Revenue, Customers, Ratings)
     - Customer segmentation
     - Category performance
     - Discount analysis

---

## Key Insights

- Discounts significantly influence customer purchasing behavior  
- Loyal customers contribute a major portion of total revenue  
- Subscription status impacts customer engagement  
- Certain product categories dominate sales performance  
- Customer demographics play a key role in revenue generation  

---

## Business Recommendations

- Optimize discount strategies to balance revenue and profit  
- Target loyal customers with personalized offers  
- Improve subscription benefits to increase engagement  
- Focus marketing efforts on high-performing categories  
- Enhance customer retention strategies
